# Iraq Taxi — iPhone starter

نموذج أولي لتطبيق راكب iPhone، عربي/English، مبني بـ SwiftUI.

## ما يحتويه

- اختيار اللغة العربية أو الإنجليزية مع اتجاه واجهة مناسب.
- شاشة طلب رحلة حديثة: الوجهة، نوع المركبة، السعر، وزر الطلب.
- تأثيرات حركة SwiftUI: طريق متحرك، نبض موقع الاستلام، وبطاقة سيارة تظهر بسلاسة.
- بيانات تجريبية فقط: لا توجد خرائط فعلية أو موقع أو دفع أو ربط بنكي في هذا النموذج.

## فتحه في Xcode

1. أنشئ مشروع **iOS App** جديداً باسم `IraqTaxi` واختَر **SwiftUI** وiOS 17 أو أحدث.
2. استبدل ملفات المشروع بالملفات داخل مجلد `IraqTaxi/`.
3. شغّل التطبيق على iPhone Simulator أو جهاز حقيقي.

## المرحلة التالية

ربط الخرائط والموقع المباشر، التسجيل برقم الهاتف، قاعدة بيانات الرحلات، ثم لوحة السائق والدفع عبر مزود عراقي رسمي بعد الحصول على عقد ومفاتيح API.
