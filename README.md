# تكسي واحد عراق | Taxi One Iraq

تطبيق لطلب سيارات الأجرة داخل العراق، عربي/English.

- **Android (جديد):** تطبيق أصلي بـ Kotlin + Jetpack Compose في مجلد [`android/`](android/) — يُبنى كملف APK.
- **iPhone:** تطبيق SwiftUI في مجلد [`IraqTaxi/`](IraqTaxi/).

## هوية المنتج

- الاسم الظاهر: **تكسي واحد عراق**
- اللغة الأساسية: العربية (RTL)، مع English.
- نطاق الخدمة: جميع محافظات العراق.
- الدفع في المرحلة الأولى: نقدي عند الوصول.

## نسخة الأندرويد (APK)

### تحميل APK جاهز

كل دفعة إلى الفرع الرئيسي تبني الـ APK تلقائياً عبر GitHub Actions:

1. افتح تبويب **Actions** في مستودع GitHub.
2. اختر آخر تشغيل ناجح لـ **Android APK**.
3. حمّل `IraqTaxi-debug-apk` (للتجربة المباشرة) أو `IraqTaxi-release-apk` (نسخة مضغوطة ومُحسّنة بـ R8).
4. انقل الملف إلى الهاتف وفعّل «التثبيت من مصادر غير معروفة» ثم ثبّته.

### البناء محلياً

يتطلب Android Studio أو JDK 17+ مع Android SDK (API 35):

```bash
cd android
./gradlew assembleDebug     # الناتج: app/build/outputs/apk/debug/app-debug.apk
```

### التوقيع للنشر

نسخة release موقّعة حالياً بمفتاح debug للتجربة فقط. قبل النشر الفعلي:

1. أنشئ مفتاحاً: `keytool -genkeypair -v -keystore release.jks -keyalg RSA -keysize 4096 -validity 10000 -alias taxione`
2. أضف `signingConfig` حقيقياً في `android/app/build.gradle.kts` يقرأ كلمات المرور من متغيرات بيئة أو `keystore.properties` (خارج Git — مضاف إلى `.gitignore`).
3. لا ترفع ملف المفتاح إلى المستودع أبداً.

## حماية المستخدم وبياناته

نسخة الأندرويد مبنية بمبدأ «أقل صلاحيات وأقل بيانات»:

- **لا إذن إنترنت إطلاقاً في هذه النسخة** — لا يمكن للتطبيق إرسال أي بيانات خارج الجهاز.
- **الموقع أثناء الاستخدام فقط**: لا يُطلب إذن الموقع في الخلفية، وتتوقف التحديثات تلقائياً عند مغادرة التطبيق أو إيقاف الزر.
- **سجل الرحلات مشفّر** بـ AES-256-GCM بمفتاح داخل Android Keystore لا يغادر العتاد الآمن.
- **لا نسخ احتياطي سحابي**: `allowBackup=false` مع قواعد استخراج بيانات تمنع نقل السجل عند تبديل الجهاز.
- **زر «حذف جميع بياناتي»** في شاشة الحساب يمسح كل ما خزّنه التطبيق.
- **حظر HTTP غير المشفّر** عبر `networkSecurityConfig` استعداداً لربط الـ Backend (مع تثبيت شهادة الخادم لاحقاً).
- **R8 minify** مفعّل في نسخة release.

## النسخة الحالية

- واجهة راكب حديثة (عربي RTL / English) مع اختيار نوع السيارة وتقدير السعر وإرسال طلب تجريبي.
- شاشة رحلات مع دورة حياة الرحلة (بحث → قبول → جارية → مكتملة/ملغاة) وسجل محفوظ محلياً.
- شاشة حساب مع تبديل اللغة وقسم الخصوصية وحذف البيانات.
- السائقون والرحلات بيانات تجريبية حتى ربط Backend آمن.

## ما قبل الإطلاق

1. Backend موثّق للركاب والسائقين والرحلات (HTTPS فقط + تثبيت شهادة).
2. تسجيل دخول برقم الهاتف (OTP) مع تخزين الرموز في التخزين المشفّر الموجود.
3. خرائط وموقع حي وإشعارات.
4. لوحة إدارة للتحقق من السائقين والدعم والتسعير.
5. اتفاق تجاري مع مزود دفع عراقي قبل إضافة الدفع الإلكتروني.
6. سياسة خصوصية وشروط استخدام وتدقيق أمني (زر طوارئ داخل الرحلة).
