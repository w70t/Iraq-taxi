# تكسي واحد عراق | Taxi One Iraq

تطبيق iPhone لطلب سيارات الأجرة داخل العراق، عربي/English، مبني بـ SwiftUI.

## هوية المنتج

- الاسم الظاهر: **تكسي واحد عراق**
- اللغة الأساسية: العربية، مع English.
- نطاق الخدمة: جميع محافظات العراق.
- الدفع في المرحلة الأولى: نقدي عند الوصول.

## النسخة الحالية

- واجهة راكب حديثة مع اختيار اللغة واتجاه RTL.
- اختيار نوع السيارة، تقدير السعر، وإرسال طلب تجريبي.
- تأثيرات حركة SwiftUI للطريق والموقع.
- لا تزال الخرائط، حسابات المستخدمين، التتبع الحي، والسائقون بيانات تجريبية حتى ربط Backend آمن.

## ما قبل الإطلاق

1. Backend موثّق للركاب والسائقين والرحلات.
2. خرائط وموقع حي وإشعارات.
3. لوحة إدارة للتحقق من السائقين والدعم والتسعير.
4. اتفاق تجاري مع مزود دفع عراقي قبل إضافة الدفع الإلكتروني.
5. سياسة خصوصية وشروط استخدام وتدقيق أمني.

## فتح المشروع في Xcode

أنشئ مشروع **iOS App** باسم `IraqTaxi`، اختر **SwiftUI** وiOS 17 أو أحدث، ثم أضف ملفات مجلد `IraqTaxi/` إلى المشروع.
